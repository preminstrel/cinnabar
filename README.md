## Usage
Clone the git repo:
```bash
git clone https://github.com/preminstrel/cinnabar.git
```

Install the packages:

```bash
pip install .
```

Run the cinnabar server:
```bash
cinnabar
```

Then visit `http://localhost:8500` in your browser.
